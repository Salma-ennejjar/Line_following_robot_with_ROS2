# Line Following Robot using ROS2

This project implements a line following robot using ROS2.

## Architecture

line_detection
Detects the line using camera or sensors.

motion_control
Computes robot motion using PID control.

status_monitor
Monitors robot state and system information.

## Technologies

- ROS2
- Python 

## Features

- Real time line detection
- PID motion control
- Robot status monitoring
